FDM + Quintessence — Extensão CLASS

Implementação de Fuzzy Dark Matter e Axion-like Quintessence no solver CLASS — Silvio A. Corrêa Júnior

Resumo: Esta extensão adiciona ao CLASS um componente de matéria escura ultraleve (FDM) e um campo de quintessência com potencial axion-like \(V(\phi)=M^4[1+\cos(\phi/f)]\). O código resolve o background e as perturbações lineares, produzindo H(z), w_\phi(z) e P(k).

w₀ (exemplo)-0.918

Instalação rápida

git clone https://github.com/talksilviojr/fdm-quintessence-class.git
cd class
# aplicar patch fornecido
patch -p1 < ../class_fdm_quintessence.patch
make
# rodar com arquivo .ini de exemplo
./class/CLASS params.ini

Nota: Veja o repositório para instruções completas e exemplos de arquivos .ini.
