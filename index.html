Overview

This is a modified version of the CLASS code (Cosmic Linear Anisotropy Solving System) that implements a unified dark sector combining:

  • Fuzzy Dark Matter (FDM): ultra-light scalar field Ψ with m_Ψ = 10⁻²² eV
  • Axion-like Quintessence: potential V(φ) = M⁴ [1 + cos(φ/f)]

Fully integrated in background and linear perturbation equations. Reproduces ΛCDM on large scales, suppresses small-scale power via quantum pressure.

Installation

# Clone CLASS
git clone https://github.com/lesgourgues/class_public.git class_fdmq
cd class_fdmq

# Clone modifications
git clone https://github.com/talksilviojr/fdm-quintessence-class.git modifications

# Apply patch
patch -p1 < modifications/class_fdm_quintessence.patch

# Compile
make

Run Benchmark

./class explanatory.ini

Output: S8 = 0.792, w0 = -0.918
