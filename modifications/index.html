Visão Geral

O FDM Quintessence Class é uma implementação avançada para modelagem de quintessência em FDM (Finite Difference Method). Esta classe fornece ferramentas e métodos para simular e analisar fenômenos de quintessência em diversos contextos físicos e matemáticos.

A quintessência é um conceito em cosmologia que se refere a uma forma hipotética de energia escura, um campo dinâmico cuja equação de estado varia com o tempo. Esta classe implementa métodos numéricos para resolver equações diferenciais relacionadas a esses fenômenos.

Instalação

Siga estas etapas para instalar e configurar o FDM Quintessence Class em seu projeto:

Clone o repositório

Comece clonando o repositório do GitHub para sua máquina local:

# Clone o repositório
git clone https://github.com/talksilviojr/fdm-quintessence-class.git
# Navegue para o diretório
cd fdm-quintessence-class

Instale as dependências

Instale as dependências necessárias usando o gerenciador de pacotes apropriado:

# Usando pip (Python)
pip install -r requirements.txt

Importe no seu projeto

Importe a classe Quintessence em seu código Python:

from fdm_quintessence import Quintessence

Como Usar

Aqui está um guia básico sobre como utilizar a classe FDM Quintessence em seus projetos:

# Importação da classe
from fdm_quintessence import Quintessence
import numpy as np

# Inicialização do modelo
model = Quintessence(
    domain_size=100,
    time_steps=500,
    initial_conditions='gaussian'
)

# Configuração de parâmetros
model.set_parameters(
    alpha=0.5,
    beta=1.2,
    potential='exponential'
)

# Execução da simulação
results = model.simulate()

# Visualização dos resultados
model.plot_evolution()
model.plot_energy_density()

Configuração de Parâmetros

A classe oferece diversos parâmetros que podem ser ajustados para diferentes cenários de simulação:

  • domain_size: Tamanho do domínio espacial para a simulação
  • time_steps: Número de passos de tempo para a evolução
  • initial_conditions: Condições iniciais do campo (gaussian, random, custom)
  • alpha, beta: Parâmetros do modelo de quintessência
  • potential: Tipo de potencial (exponential, power_law, custom)

Simulação Básica de Quintessência

Este exemplo demonstra uma simulação básica com condições iniciais gaussianas:

# Exemplo de simulação básica
from fdm_quintessence import Quintessence
import matplotlib.pyplot as plt

# Criar e configurar o modelo
model = Quintessence(domain_size=200, time_steps=1000)
model.set_parameters(alpha=0.7, beta=1.5)

# Executar simulação
results = model.simulate()

# Visualizar evolução temporal
model.plot_evolution()
plt.title('Evolução do Campo de Quintessência')
plt.show()

Simulação com Potencial Personalizado

Este exemplo mostra como usar um potencial personalizado para a quintessência:

# Exemplo com potencial personalizado
from fdm_quintessence import Quintessence
import numpy as np

# Definir função de potencial personalizado
def custom_potential(phi):
    return 0.5 * phi**2 + 0.25 * phi**4

# Configurar modelo com potencial personalizado
model = Quintessence()
model.set_parameters(potential=custom_potential)

# Executar simulação e visualizar resultados
results = model.simulate()
model.plot_potential()
model.plot_phase_diagram()

Análise de Densidade de Energia

Este exemplo foca na análise da densidade de energia durante a evolução:

# Análise de densidade de energia
from fdm_quintessence import Quintessence

# Configurar modelo para análise detalhada
model = Quintessence(
    domain_size=150,
    time_steps=800,
    initial_conditions='random'
)

# Executar simulação com saída detalhada
results = model.simulate(verbose=True)

# Analisar componentes de energia
kinetic_energy = model.calculate_kinetic_energy()
potential_energy = model.calculate_potential_energy()
total_energy = model.calculate_total_energy()

# Plotar evolução das energias
model.plot_energy_components()

Referência da API

Documentação completa dos métodos e parâmetros disponíveis na classe Quintessence:

Métodos Principais

__init__(domain_size=100, time_steps=500, initial_conditions='gaussian')

Inicializa uma nova instância da classe Quintessence.

  • domain_size: Tamanho do domínio espacial (padrão: 100)
  • time_steps: Número de passos de tempo (padrão: 500)
  • initial_conditions: Tipo de condições iniciais ('gaussian', 'random', 'custom')

set_parameters(alpha=1.0, beta=1.0, potential='exponential')

Configura os parâmetros do modelo de quintessência.

  • alpha: Parâmetro alpha do modelo (padrão: 1.0)
  • beta: Parâmetro beta do modelo (padrão: 1.0)
  • potential: Função de potencial ('exponential', 'power_law' ou função personalizada)

simulate(verbose=False)

Executa a simulação do modelo de quintessência.

  • verbose: Se True, exibe informações de progresso (padrão: False)
  • Retorna: Array com os resultados da simulação

plot_evolution()

Gera um gráfico da evolução temporal do campo de quintessência.

plot_energy_density()

Gera um gráfico da densidade de energia ao longo do tempo.

Métodos de Análise

  • calculate_kinetic_energy(): Calcula a energia cinética do campo
  • calculate_potential_energy(): Calcula a energia potencial do campo
  • calculate_total_energy(): Calcula a energia total do campo
  • get_equation_of_state(): Calcula o parâmetro da equação de estado
