Modelo Unificado do Setor Escuro

Esta é uma versão modificada do CLASS que implementa um setor escuro duplo com dois campos escalares:

  • Fuzzy Dark Matter (FDM): Campo escalar ultra-leve Ψ (m_Ψ = 10⁻²² eV)
  • Quintessência tipo Axion: Energia escura dinâmica com V(φ) = M⁴ [1 + cos(φ/f)]

Totalmente integrado em fundo e perturbações lineares. Reproduz ΛCDM em grandes escalas, suprime formação de estrutura via pressão quântica.

Instalação

# Clone CLASS público
git clone https://github.com/lesgourgues/class_public.git class_fdmq
cd class_fdmq

# Clone modificações
git clone https://github.com/talksilviojr/fdm-quintessence-class.git modifications

# Aplicar patch
patch -p1 < modifications/class_fdm_quintessence.patch

# Compilar
make

Executar Benchmark

./class explanatory.ini

Saída: S₈ = 0.792 | w₀ = -0.918
